## 📱 반응형 브레이크포인트

### GitHub Pages
- **1024px 이하**: 중간 화면 최적화
- **768px 이하**: 태블릿/모바일
  - 헤더 세로 정렬
  - 네비게이션 세로 배치
  - 전체 너비 버튼
- **480px 이하**: 초소형 모바일
  - 작은 폰트 크기
  - 최소 패딩

### MicroBoard
- **1024px 이하**: 패딩 조정
- **768px 이하**: 모바일 레이아웃
  - 헤더 세로 정렬
  - 네비게이션 중앙 정렬
  - 사용자 정보 세로 배치
- **480px 이하**: 초소형 화면
  - 작은 폰트
  - 최소 간격
