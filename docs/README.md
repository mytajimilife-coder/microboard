# GitHub Pages Documentation

This directory contains the static content for the MicroBoard GitHub Pages site.

## How to Enable GitHub Pages

1.  Go to your repository settings on GitHub.
2.  Navigate to the "Pages" section.
3.  Under "Source", select "Deploy from a branch".
4.  Under "Branch", select your main branch and the `/docs` folder.
5.  Click "Save".

Your site will be published at `https://yourusername.github.io/microboard/`.
